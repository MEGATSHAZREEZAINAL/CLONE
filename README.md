# Dart AI Website Clone

This is a clone of the [Dart AI](https://www.dartai.com/) website, created for educational purposes. The clone demonstrates modern web development techniques and responsive design principles.

## Features

- Responsive design that works on mobile, tablet, and desktop
- Modern CSS with variables for consistent styling
- Interactive elements using JavaScript
- Multiple pages including home, blog, and integrations

## Pages Included

1. **Home Page (index.html)** - Main landing page with features, solutions, and testimonials
2. **Blog Page (blog.html)** - Blog listing with article previews
3. **Integrations Page (integrations.html)** - Showcase of integration options

## Technologies Used

- HTML5
- CSS3 (with custom properties/variables)
- JavaScript (vanilla)
- Font Awesome for icons

## How to Use

1. Clone this repository
2. Open any of the HTML files in your browser to view the pages
3. Explore the code to understand the structure and styling

## Structure

```
├── index.html          # Home page
├── blog.html           # Blog listing page
├── integrations.html   # Integrations page
├── css/
│   └── styles.css      # Main stylesheet
├── js/
│   └── main.js         # JavaScript functionality
└── README.md           # Project documentation
```
